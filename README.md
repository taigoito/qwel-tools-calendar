# Business Calendar | Created by ― QWEL.DESIGN

営業日カレンダー。  
民宿の空室状況管理のための簡易システムとして開発。  

SQLiteで状態管理 → 自作APIからfetch → JSで描画 と、  
PHP / Vanilla JS でスクラッチした自信作のひとつ。
